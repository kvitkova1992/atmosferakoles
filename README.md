# Атмосфера Колёс — Сайт детейлинг-центра

Современный многостраничный сайт для детейлинг-центра "Атмосфера Колёс" (Пермь).

## Технологии

- React 19 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- HashRouter

## Страницы

- **Главная** — Hero, этапы работ, преимущества, призыв к действию
- **Портфолио** — 8 секций с фото и видео работ
- **Услуги** — 6 разделов: реставрация, правка, суппорта, шиномонтаж, детейлинг, пескоструй
- **Прайс** — актуальные цены
- **Партнёры** — форма для партнёров
- **Контакты** — адрес, телефон, соцсети
- **Политика ПДн** — политика конфиденциальности

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.
